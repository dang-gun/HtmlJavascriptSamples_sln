# visual studio Standalone Vite + Typescript Project

## 개요
visual studio 2022에서 Vite + Typescript로 설정된 셈플입니다.

- '.esproj'(자바스크립트 프로젝트 시스템) 템플릿으로 생성되었습니다.
- Vite를 개발 서버(Dev Server) 및 빌더(builder)로 사용합니다.
- Typescript로 구성되어 있습니다.
